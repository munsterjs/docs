Commands

Here is a list of available commands for munster cli:

CommandSyntaxDescription
newmster new <project_name>Generate a new munster application.
docsmster docsOpens the official munster framework documentation.
generatemster generate <type> <name>Generate a munster file based on the given type and name. See generate types section for more info.

Generate types#

Here's a list of available types for the generate command:

TypeDescription
componentGenerate a component file.
middlewareGenerate a middleware file.
moduleGenerate a module file.

Here's an example on how to use the generate command:

mster generate component sidebar

This will generate a component file in src/sidebar.component.html.

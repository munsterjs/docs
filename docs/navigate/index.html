Navigate

Navigate allows us to redirect to a route programmatically.

Register the plugin#

Navigate plugin needs to be registered in a module before we can use it. To register this plugin we just need to pass it to the plugins array of a module.

After the this plugin is registered to the module. All Components under the module will have this.$navigate method.

Here's an example on how to register this plugin:

import { Module, Navigate } from '@munster/core';
@Module({
plugins = [
Navigate
];
})
export class AppModule { }

Navigate programmatically#

Here's an example on how to navigate to a route programmatically:

import Template from './sample.component.html';
import { Component, SuperClass } from '@munster/core';
@Component({
selector: 'sample-component',
template: Template
})
export class SampleComponent extends SuperClass() {
btnClick() {
this.$navigate('/route/path');
}
}

Navigate parameters#

Navigate has three parameters, first is the url, second is the state and third is the title. The second and third parameters are optional.

For more information on url, state and title please follow this link at developer.mozilla.org.
